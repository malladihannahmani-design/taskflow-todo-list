# TaskFlow – Smart To-Do List Web Application

## 📌 Project Overview

* Interactive To-Do List web application
* Built using HTML, CSS, and JavaScript
* Helps users manage daily tasks efficiently
* Responsive and colorful modern UI

---

# 🚀 Main Features

* Add new tasks
* Edit existing tasks
* Delete tasks
* Mark tasks as completed
* Filter tasks:

  * All
  * Active
  * Completed
* Data saved using localStorage
* Responsive design for mobile and desktop

---

# 🛠 Technologies Used

* HTML5

# ⚙️ How the Project Works

## 1️⃣ Add Task

Users can enter a task and add it to the list.

## 2️⃣ Update Task

Existing tasks can be edited.

## 3️⃣ Delete Task

Tasks can be removed permanently.

## 4️⃣ Complete Task

Tasks can be marked as completed using checkbox.

## 5️⃣ Filter Tasks

Users can view:

* All Tasks
* Active Tasks
* Completed Tasks

## 6️⃣ Save Data

Tasks remain saved even after refreshing browser using localStorage.
